# Pandas

Pandas is built on top of NumPy and provides the **DataFrame**, a structure inspired by R and Excel tables.

## 1. Creating DataFrames

```python
import pandas as pd

data = {
    "feature1": [1.0, 2.1, 3.5],
    "feature2": [0.5, 0.2, 0.9],
    "label": [0, 1, 0]
}

df = pd.DataFrame(data)
print(df.head())
```

```python
print(df.info())      # Types and missing values
```

## 5. Data Cleaning

AI models require clean data.

